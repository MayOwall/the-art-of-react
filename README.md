## 프로젝트 목적

이 프로젝트는 리액트를 다시 차근차근 학습하는 것에 의의를 두고 있습니다.

기존에 필요할 때만 급급하게 임시방편으로 학습하던 방식과는 달리 전반적인 지식을 학습하여 <br/>
필요한 상황에 적합한 로직을 빠르게 적용할 수 있도록 성장하는 것을 목표하고 있습니다.

<br/>
